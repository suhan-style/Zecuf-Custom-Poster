Demo: Responsive Zecuf Custom Poster

Ref: Demo video is created for more understanding...


Technology
===========
HTML
CSS
Javascript
Jquery
PHP


Process
=========
1. choose frame
2. upload any photo to display in a frame
3. adjust photo by Drag Image button
4. choose image to display at bottom of your poster
5. download your custom poster


Feature
==================
1. admin can view users custom poster (ie. by clicking on download, clone of that poster image will be saved on server)


Note
======
-- This project require (SERVER) to perform functionality.
-- This will work on (localhost) also.
